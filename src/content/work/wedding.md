### Application de gestion des invités de mariage

Le projet **RSVP Wedding** est une application web développée avec Next.js, permettant de gérer les réponses des invités à un mariage. Elle propose un formulaire RSVP pour les invités, un tableau de bord d’administration sécurisé, et une interface responsive adaptée à tous les appareils.

>Structure et Architecture

Plateforme fonctionnelle permettant aux invités de confirmer leur présence et aux organisateurs de suivre les réponses en temps réel.  
Utilisation de Next.js pour le routage, le rendu côté serveur et la structure du projet.

Structure **moderne React** avec composants réutilisables et hooks personnalisés.  
Connexion à **Firebase** pour la gestion de la base de données (Firestore) et de l’authentification.

Interface d’administration protégée par authentification, permettant de visualiser et gérer les réponses.

*Technologies utilisées*  
* Next.js : Pour le framework frontend et le routage.  
* Tailwind CSS : Pour le design et la responsivité.  
* Firebase : Pour la base de données et l’authentification.  
* React : Pour la création des interfaces utilisateur dynamiques.
* Vercel : Pour le déploiement et l’hébergement de l’application.
